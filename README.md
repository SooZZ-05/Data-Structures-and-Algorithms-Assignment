# Clinic Management System

A Java-based clinic management system developed for the **BMCS2063 Data Structures and Algorithms** assignment. This project applies custom Abstract Data Types (ADTs) and data structures to support clinic operations such as patient registration, doctor scheduling, consultations, treatments, and pharmacy inventory management.

## Overview

This system was designed to support multiple clinic roles, including:

- Patient
- Doctor
- Receptionist
- Pharmacist :contentReference

The system consists of several modules:

- Patient Management
- Doctor Management
- Consultation Management
- Medical Treatment Management
- Pharmacy Management :contentReference
